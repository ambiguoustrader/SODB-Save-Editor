# Contributing

Спасибо за интерес к проекту.

## Как предложить изменение

1. Создай отдельную ветку.
2. Не добавляй реальные `.sodb` сейвы, JSON-экспорты и персональные данные.
3. Проверь запуск:

```bat
python -m pip install -r requirements.txt
python sod_save_editor.py
```

4. Для сборки Windows EXE:

```bat
build_exe.bat
```

## Стиль изменений

- Не усложняй зависимости без необходимости.
- Для GUI предпочтительно использовать стандартный `tkinter`.
- Любые поля сейва, которые не до конца понятны, помечай осторожно: «вероятно», «наблюдается», «может отличаться между версиями».
