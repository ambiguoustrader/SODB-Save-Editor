# Security Policy

Это локальный фанатский инструмент для редактирования сохранений игры.

## Рекомендации

- Не публикуй реальные сейвы, если в них есть данные, которые не хочешь раскрывать.
- Не открывай файлы из источников, которым не доверяешь.
- Перед перезаписью сохранений делай бэкап.

## Сообщения о проблемах

Если нашёл баг, из-за которого сейв портится, опиши:

- версию игры;
- версию редактора;
- действие, после которого возникла проблема;
- по возможности — минимальный обезличенный пример структуры JSON.

Не прикладывай полный личный сейв публично, если не уверен, что он не содержит чувствительной информации.
